Design Phase 1 of “PUP: Campus Creatives,” a responsive university creative portfolio and showcase interface for the Polytechnic University of the Philippines.

This phase is for interface design only.

Do not build:

- Backend
- Database
- Authentication logic
- API routes
- File upload logic
- Real form submission
- Admin functionality
- Moderator functionality
- Full Explore, Gallery, Events, or Creator Profile pages

Use static sample data and front-end-only prototype interactions.

Create both:

1. Desktop website interface
2. Mobile-responsive interface

The desktop and mobile versions must use the same visual system and content, but the mobile version must be properly adapted instead of simply shrinking the desktop layout.

==================================================
PROJECT PURPOSE
==================================================

Campus Creatives is a university-based creative portfolio and showcase platform for PUP students.

It allows students to:

- Discover creative works
- Build a campus creative identity
- View featured student creators
- Find events and open calls
- Gain recognition through highlights and badges
- Organize creative works in a more lasting way than ordinary social media

The platform should feel like:

- A PUP campus creative hub
- A student portfolio platform
- A university gallery
- A curated campus showcase
- A modern academic creative community

It should not feel like:

- Facebook
- TikTok
- A generic social media feed
- A corporate dashboard
- A luxury art marketplace
- A generic startup landing page

The interface must be portfolio-first, organized, supportive, and clearly connected to the PUP campus community.

PUP-INSPIRED VISUAL IDENTITY
==================================================

Use a visual theme strongly based on the PUP maroon and gold color family.

Primary colors:

- PUP Maroon: #800000
- Deep Maroon: #5B0000
- Dark Maroon: #3D0000
- PUP Gold: #FFB81C
- Warm Gold: #D89B00
- Soft Gold: #FFF2CC

Supporting red:

- Crimson Accent: #A71930
- Soft Maroon Surface: #F5E7E7

Neutral colors:

- Main Background: #F8F6F1
- Warm White: #FFFDF8
- Card Background: #FFFFFF
- Secondary Surface: #F1EDE6
- Primary Text: #211B1B
- Secondary Text: #665D5D
- Muted Text: #8A8080
- Border: #DDD5CC
- Dark Surface: #251717

Status colors:

- Approved: #2F7D57
- Pending: #B7791F
- Needs Revision: #C76A24
- Rejected: #B63737
- Information: #356B9C

Color rules:

- Maroon is the main brand and action color.
- Gold is used for recognition, selected states, highlights, official labels, and featured content.
- Use warm white or cream backgrounds instead of cold gray.
- Do not use purple or blue as the main theme.
- Do not cover entire pages in maroon.
- Do not overuse bright gold.
- Maintain readable contrast.
- Never use gold body text on white.

==================================================
DESIGN STYLE
==================================================

Use a clean, modern, academic, and creative visual style.

The design should combine:

- PUP campus identity
- Portfolio layouts
- Gallery presentation
- Editorial campus features
- Student-friendly navigation

Use:

- Large media previews
- Warm backgrounds
- Clean cards
- Thin borders
- Soft shadows
- Strong typography hierarchy
- Clear section spacing
- Maroon highlights
- Gold recognition badges
- Rounded but not overly playful shapes
- Editorial featured sections
- Campus-related visual details

Avoid:

- Excessive gradients
- Glassmorphism
- Neon colors
- Heavy shadows
- Cartoon-like illustrations
- Oversized decorations
- Dense interfaces
- Endless vertical social feeds
- Floating elements without purpose

Corner radius system:

- Small controls: 8px
- Inputs and buttons: 10–12px
- Standard cards: 14–16px
- Featured panels: 18–20px

Use subtle shadows only.

==================================================
HCI AND ACCESSIBILITY
==================================================

Follow these requirements:

- Clear navigation
- Strong content hierarchy
- Consistent controls
- Visible active states
- Readable contrast
- Visible keyboard focus
- Clear form labels
- Error messages near fields
- Do not rely only on color
- Use icons with labels for important actions
- Maintain readable mobile text
- Use descriptive alt text
- Keep buttons and links predictable
- Keep repeated actions in consistent positions
- Use supportive wording such as “Appreciate” instead of “Like”
- Make recognition visible without making the platform overly competitive
