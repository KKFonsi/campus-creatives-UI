PHASE 3 PURPOSE
==================================================

Phase 3 focuses on the student creator experience.

- View a complete creator profile
- Edit their creator information
- Organize a portfolio
- Submit a creative work
- Submit an entry to an event
- Review a submission before sending it
- Track moderation status
- Read moderator feedback
- Revise a submission
- View saved works
- Manage portfolio visibility
- Share their creator profile

The design must remain:

- Portfolio-first
- Campus-centered
- Moderated
- Organized
- Supportive
- Professional
- Student-friendly

It must not become:

- A generic social media profile
- A commercial portfolio marketplace
- A content creator dashboard
- A cluttered upload system
- A corporate form portal

==================================================
HCI AND ACCESSIBILITY
==================================================

Follow these principles:

- Clearly show progress in multi-step forms.
- Keep one main task per step.
- Show required and optional fields.
- Place validation near the affected field.
- Allow users to save drafts.
- Warn users before leaving with unsaved changes.
- Clearly explain moderation status.
- Do not use hostile or punitive feedback language.
- Keep moderator feedback actionable.
- Use icons and labels for all statuses.
- Use visible keyboard focus states.
- Maintain minimum 44px mobile touch targets.
- Use descriptive image alt text.
- Provide caption and transcript fields where relevant.
- Use clear confirmation before destructive actions.
- Keep repeated actions in consistent locations.
- Preserve entered values when moving between prototype steps.
- Keep portfolio organization understandable.
- Avoid overemphasizing follower or appreciation counts.
