==================================================
PHASE 2 PURPOSE
==================================================

Phase 2 focuses on public creative discovery.

Users should be able to:

- Browse approved student creative works
- Search and filter works
- Open a complete creative work or project
- Browse curated campus exhibitions
- Discover events and open calls
- Open detailed event information
- Browse student talent by college
- View short-form creative Spotlight content
- Appreciate, save, comment on, share, and report content through visual prototype interactions

The design must remain:

- Portfolio-first
- Media-focused
- Organized
- Campus-centered
- Moderated
- Supportive
- Professional enough for student portfolios

It must not become:

- A generic social media feed
- A TikTok clone
- A Facebook clone
- A commercial art marketplace
- A corporate content-management interface

==================================================

- Clearly display active filters.
- Allow filters to be removed individually.
- Show result counts.
- Make event deadlines prominent.
- Clearly distinguish official and student content.
